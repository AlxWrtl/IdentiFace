## 🚀 Installation

### 1️⃣ Cloner le projet

```bash
git clone https://github.com/AlxWrtl/IdentiFace.git
cd IdentiFace
```

### 2️⃣ Créer un environnement virtuel avec `uv`

```bash
uv venv .venv
source .venv/bin/activate
```

### 3️⃣ Installer les dépendances

```bash
uv pip install -r requirements.txt
```
